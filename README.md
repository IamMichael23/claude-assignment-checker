# Claude Assignment Checker Skill

A Claude Code superpowers skill for comprehensive pre-submission assignment checking using a 3-agent team approach.

## What It Does

Automatically reviews academic work before submission using three specialist agents:
- **Academic Critic** — evaluates argument quality, logic, and academic rigor
- **Rubric Grader** — checks against assignment requirements and grading criteria
- **Copy Editor** — proofreads for grammar, style, and formatting issues

## Triggers

Activates when you say things like:
- "check my assignment"
- "review before submitting"
- "is this ready to hand in?"
- "grade my work"
- "how can I improve my grade?"

## Installation

Place the `assignment-checker/` folder in your Claude Code superpowers skills directory.
